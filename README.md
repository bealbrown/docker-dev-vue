# docker-dev-vue

> A Vue.js project

## Run develop

```bash
# docker run --rm -it -v $(pwd):/src -w /src -e HOST=0.0.0.0 -p 8080:8080 node:8.9.4 bash -c "npm install && npm run dev"
```

## Run with docker compose

```bash
# docker-compose up
```
